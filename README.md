# chanks.dev - Personal Portfolio Website

A modern, animated portfolio website built with React, featuring a galaxy animation background, interactive development kit showcase, and sleek component designs.

## 🌟 Features

- Interactive galaxy background animation with parallax effects
- Responsive header with animated Oni Torii Gate design
- Sleek development kit showcase with hover animations
- Premium about section with modern UI elements
- Dark theme with professional color scheme
- Mobile-responsive design
- Smooth animations and transitions using Framer Motion

## 🚀 Tech Stack

- React
- Tailwind CSS
- Framer Motion
- Lucide Icons
- React Router

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Node.js (v16.0.0 or higher)
- npm (v7.0.0 or higher)
- Git

## ⚙️ Installation

1. Clone the repository (private access required)
```bash
git clone https://github.com/YourUsername/chanks.dev.git
cd chanks.dev
```

2. Install dependencies
```bash
npm install
```

3. Create a `.env` file in the root directory and add any necessary environment variables

4. Start the development server
```bash
npm run dev
```

5. Build for production
```bash
npm run build
```
